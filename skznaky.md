# Dôležité znaky na slovenskej klávesnici

| Znak | Názov | Klávesová skratka |
|------|-------|-------------------|
| `@` | Zavináč | **Alt Gr + V** (alebo **Ctrl + Alt + V**) |
| `_` | Podčiarkovník (underscore) | **Alt Gr + Q** (alebo **Shift + -**) |
| `\|` | Zvislá čiara (pipe) | **Alt Gr + W** (alebo **Ctrl + Alt + W**) |
| `~` | Tilda (vlnovka) | **Alt Gr + 1** (alebo **Ctrl + Alt + 1**) |
| `{` | Ľavá zložená zátvorka | **Alt Gr + B** (alebo **Ctrl + Alt + B**) |
| `}` | Pravá zložená zátvorka | **Alt Gr + N** (alebo **Ctrl + Alt + N**) |
| `[` | Ľavá hranatá zátvorka | **Alt Gr + F** (alebo **Ctrl + Alt + F**) |
| `]` | Pravá hranatá zátvorka | **Alt Gr + G** (alebo **Ctrl + Alt + G**) |
| `#` | Mriežka (hash) | **Alt Gr + X** (alebo **Ctrl + Alt + X**) |
| `&` | Ampersand (gauč) | **Alt Gr + C** (alebo **Ctrl + Alt + C**) |
| `*` | Hviezdička | **Alt Gr + /** (alebo **Ctrl + Alt + /**) |
| `<` | Ľavá ostrá zátvorka | **Alt Gr + ,** (alebo **Ctrl + Alt + ,**) |
| `>` | Pravá ostrá zátvorka | **Alt Gr + .** (alebo **Ctrl + Alt + .**) |
| `$` | Dolár | **Alt Gr + 3** (alebo **Ctrl + Alt + 3**) |
| `€` | Euro | **Alt Gr + E** (alebo **Ctrl + Alt + E**) |
| `\` | Spätná lomka (backslash) | **Alt Gr + Q** niekedy, alebo **Alt Gr + ú** (pravá klávesa vedľa Enter) |
| `` ` `` | Spätný apostrof (backtick) | **Alt Gr + 7** (alebo **Ctrl + Alt + 7**) |
| `%` | Percento | **Shift + 5** |
| `/` | Lomka | **Shift + ý** (alebo klávesa vedľa pravého Shiftu) |
| `=` | Rovná sa | **Shift + 1** |
| `+` | Plus | **Shift + 2** |
| `'` | Apostrof | **Shift + 4** |
| `"` | Úvodzovky | **Shift + 6** |
| `!` | Výkričník | **Shift + 9** |
| `?` | Otáznik | **Shift + ,** |
| `:` | Dvojbodka | **Shift + .** |
| `;` | Bodkočiarka | **Shift + ;** |

> **Poznámka:** **Alt Gr** je pravý kláves **Alt** na slovenskej klávesnici. V niektorých programoch alebo prostrediach môže fungovať aj kombinácia **Ctrl + Alt** namiesto **Alt Gr**.
